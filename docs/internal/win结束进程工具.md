# Geek Killer Pro - 极客级 Windows 进程强制终止工具

这是基于 **Rust + egui + eframe** 打造的现代化 GUI 进程管理工具。专为解决电脑卡顿、软件异常、高负载场景设计，具有极致的性能和极简的极客视觉风格。

## 核心特性

1.  **极低资源占用**：采用智能重绘控制，静止时 CPU 占用接近 0%，即使在电脑极度卡顿的情况下也能顺滑运行。
2.  **智能诊断报告**：一键识别僵死进程（DEAD）、严重高负载任务（CPU > 80%）及内存大户，并提供处理建议。
3.  **智能归类显示**：
    *   **极高负载任务**：自动提取最耗资源的进程置顶显示。
    *   **活动用户任务**：清晰展示当前用户的应用。
    *   **系统核心服务**：安全折叠系统关键进程，防止误杀。
4.  **一键终止全组**：支持终止进程及其所有关联子进程，彻底清理残留。
5.  **极客视觉风格**：采用深咖啡黑与复古金配色，无乱码设计，支持 4K 高清缩放。
6.  **管理员模式检测**：自动识别权限状态，确保拥有最底层的进程控制权。

## 技术架构

*   **语言**: Rust (2021 Edition)
*   **GUI 框架**: `eframe` + `egui` (即时模式 GUI，极致响应)
*   **系统监控**: `sysinfo v0.30.13` (原生跨平台系统接口)
*   **底层接口**: `windows-sys` (直接调用 Win32 API 进行权限校验)
*   **资源处理**: `image` (图标处理) & `winres` (编译嵌入图标)

## 开发与编译说明

### 1. 环境准备
确保已安装 Rust 环境。如果未安装，请访问 [rust-lang.org](https://www.rust-lang.org/) 安装。

### 2. 项目结构
*   `geek_killer/src/main.rs`: 核心业务逻辑与 GUI 实现。
*   `geek_killer/Cargo.toml`: 依赖与项目配置。
*   `geek_killer/build.rs`: Windows 资源编译（图标嵌入）。
*   `进程图标.ico`: 资源管理器显示的 EXE 图标。
*   `进程图标.png`: 程序运行时窗口显示的图标。

### 3. 编译发布
在 `geek_killer` 目录下执行以下命令：

```powershell
# 检查代码规范
cargo clippy --release

# 编译最终优化版本 (EXE 在 target/release 目录下)
cargo build --release
```

## 极客开发总结与经验

1.  **性能与响应的平衡**：在即时模式 GUI (egui) 中，通过 `request_repaint_after` 替代持续重绘，是降低 CPU 占用的关键技巧。
2.  **乱码防御**：在 Windows 环境下，尽量避免在代码中直接使用非标准 Emoji，改用纯文本或矢量符号，以确保不同系统版本的兼容性。
3.  **资源嵌入**：使用 `include_bytes!` 配合 `image` 库，可以将图标直接编译进二进制文件，实现真正的“单文件绿色版”。
4.  **分类逻辑**：将进程按资源消耗而非单纯的字母顺序排列，能极大提高用户在紧急卡顿情况下的操作效率。
