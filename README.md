# 🚀 Geek Killer Pro (极客杀手) v2.0

> **"不仅仅是杀进程，更是系统的最后一道防线。"**

Geek Killer Pro 是一款专为 Windows 极客打造的高性能系统管理工具。它摒弃了传统任务管理器的繁琐，专注于**极速响应**、**暴力解压**和**硬件安全弹出**。

![界面预览](Resources/重构版界面截图.png)

## ✨ 核心功能

### 1. 💾 暴力 U 盘/硬盘弹出 (U-Disk Terminator)
解决 Windows 最令人头疼的 *"该设备正在使用中"* 问题。
- **秒开模式**：点击弹出即刻响应，无扫描延迟。
- **智能破防**：自动检测占用进程（包括 Restart Manager 漏掉的）。
- **强力清场**：一键杀进程 + 强制卸载卷 (Dismount) + 弹出父设备。
- **小白友好**：自动翻译晦涩的 Win32 错误代码（如 `VetoType 6` -> "系统核心组件锁定"）。

### 2. ⚡ 极简模式 (Panic Mode)
当你的电脑卡死（CPU 100% / 内存耗尽）时，它是你唯一的救星。
- **自适应调度**：检测到高负载时，自动降低自身资源占用（刷新率降至 0.5Hz）。
- **界面降级**：自动折叠非必要面板，只显示导致卡顿的罪魁祸首。
- **非阻塞架构**：即使系统再卡，UI 也能保持响应（基于 `try_read` 无锁设计）。

### 3. 📊 极客遥测 (Geek Telemetry)
- **精准识别**：自动识别国产软件（微信、钉钉、火狐、豆包等）的中文名称。
- **分类清晰**：将进程分为 `🔥 极高负载`、`👤 用户应用`、`🛡️ 系统服务`，一目了然。
- **实时监控**：CPU、内存、网络、磁盘 I/O 实时刷新。

## 📦 安装与使用

1.  **下载**：获取最新发布的 `极客杀手_Pro_v2.0_绿色版.zip`。
2.  **运行**：解压后直接运行 exe（建议右键 "以管理员身份运行" 以获得完整杀进程权限）。
3.  **便携**：纯绿色软件，无注册表残留，即用即走。

## 🛠️ 技术栈
- **语言**：Rust 🦀 (安全、高效)
- **GUI**：egui (即时模式 GUI，轻量级)
- **核心 API**：Win32 API (RestartManager, SetupAPI, Ioctl)
